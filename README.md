## STAR SHARE v1.0.0 - Peer-to-Peer Local File Sharing & Chat 🚀

We are excited to announce the first official release of **STAR SHARE**, a cross-platform, local network utility developed in Flutter that allows you to instantly share files, directories, and chat messages between Android and Windows devices completely offline.

### 🌟 Key Features & Updates in this Release:

* **100% Automated Subnet Scanning:** Replaces unreliable UDP broadcast with a highly robust TCP subnet scanner. Works seamlessly even on strict mobile hotspots, guest networks, and enterprise Wi-Fi routers.
* **Direct P2P Sockets:** Achieves high-speed local data transfer by establishing direct TCP socket streams.
* **Multi-File & Folder Transfer:** Seamlessly queue and send multiple files or entire directories at once.
* **Smart Local Chat Room:** Send public or private text messages. Messages can be encrypted with a 4-digit PIN (shown as `****` 🔒 to others) or hidden with an Eye icon (👁️) toggle for public viewing control.
* **Direct Downloads Integration:** Android files save directly to the system's public `/Download` folder. Windows files save to the standard `Downloads` directory.
* **Native File Explorer Launcher:** Launch your system's native downloads or file manager directly from the application's history tab.
* **Persistent Transfer History:** A standalone history tab keeps a clean log of all file transfers (incoming/outgoing, size, speed, timestamp, and sender/receiver name).
* **Hexa-lingual Localization:** Instant, real-time translations for English, Persian, Arabic, Spanish, Chinese, and Japanese.
* **Dark & Light Themes:** Adaptive modern user interface supporting system themes.
* **Automated Update Checker:** Automatically fetches the latest version information from GitHub on startup and provides direct redirection.

### ⚙️ How to install:
1. **Android:** Download the `.apk` asset and install it on your mobile device.
2. **Windows:** Download the zipped `Release` folder, extract it, and run `.exe` asset.
